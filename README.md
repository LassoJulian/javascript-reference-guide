# JavaScript Reference Guide

Guía interactiva de JavaScript con foco en práctica, casos reales y progresión pedagógica.

## Contenido

- Fundamentos, control de flujo, colecciones y objetos
- Funciones, closures, errores y asincronía
- Módulos, DOM, APIs del navegador y patrones avanzados
- Testing, debugging y performance
- Práctica UI con todo app, validación de formularios y delegación dinámica
- Eventos con listeners, propagación, formulario demo y ejemplos de `fetch`

## Uso

1. Abre `index.html` en el navegador.
2. Navega por el índice lateral.
3. Ejecuta las demos del laboratorio interactivo.

## Estructura

```text
index.html   - Guía principal
styles.css   - Sistema visual Deep Learning High-Tech
script.js    - Demos, navegación y laboratorio interactivo
LICENSE      - MIT

## Incluye

- Laboratorio interactivo con demos ejecutables
- Mini todo app con persistencia local
- Formulario de ejemplo con `input`, `change`, `submit` y `reset`
- Demo de `fetch` con `AbortController`
- Demo de `removeEventListener`
```

## Enfoque didáctico

- Cada sección incluye conceptos, caso de uso y ejemplo
- Se priorizan patrones de producción sobre sintaxis aislada
- El contenido está ordenado de lo básico a lo avanzado
